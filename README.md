# webappzz website

Static company website for `webappzz`, ready for GitHub hosting and domain connection.

## Files

- `index.html` - main landing page
- `works.html` - project references with screenshot placeholders
- `domains.html` - separate domains page
- `privacy.html` - privacy policy page
- `styles.css` - shared design system and responsive styles
- `script.js` - simple reveal animation

## Quick edits

- Update package text in `index.html`
- Replace project placeholders in `works.html`
- Replace domain chips in `domains.html`
- Adjust privacy policy text in `privacy.html` if needed
- Add your real logo image later if you want to replace the text-based brand mark
